# Arquétipos — JogoCarta

Regra 3+2: cada arquétipo tem 3 mecânicas principais + 2 secundárias.

| Arquétipo | ID DB | Recurso Especial | 3 Mecânicas Principais | 2 Secundárias |
|-----------|-------|-----------------|------------------------|---------------|
| Anjos | angels | faith | Escudo Divino, Fé, Invocação do Extra Deck | Elo Vital, Bastião |
| Demônios | demons | - | Oferenda, Exumar, Sacrifício | Drenar, Último Suspiro |
| Dragões | dragons | treasure | Atropelar, Sobrecarga, Tesouro | Amedrontar, Supremacia |
| Abissais | abyssals | - | Fluxo da Maré, Avanço/Retirada, Dragar/Vaticinar | Silêncio, Veneno |
| Mecânicos | mechanics | - | Artefatos, Estruturas, Simbiose | Consumir, Radar |
| Pútridos | undead | - | Último Suspiro, Eco, Exumar | Regenerar, Herança |
| Goblins | goblins | - | Tributo, Tokens, Agressão | Infestação, Vingança |
| Espectros | specters | - | Sombra, Infundir, Penetrar defesas | Sifão, Canalizar |
| Aranhas | spiders | - | Aprisionar/Casulo, Infestação/Filhotes, Veneno | Tocaia/Emboscada, Teia/Radar |
| Sombrios | shadows | - | Roubo, Emboscada, Gancho | Sombra, Furtividade |

**Why:** Referência rápida para implementar mecânicas de cada arquétipo.
**How to apply:** Ao criar cartas ou implementar efeitos, verificar se a mecânica pertence ao arquétipo correto e segue a Regra 3+2.
